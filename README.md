# Acree Lab Calculator 🧪🍪

A single-file, offline spectrophotometry calculator for the Acree Lab (Beer-Lambert standards + unknowns).

- Standards → epsilons, automatically
- Dilution Factor Finder
- Unknowns → mass percent, with auto epsilon-matching
- Built-in help button with the full lab procedure
- Works fully offline — just open `index.html` in any browser

Made by Ananya for Dr. Acree 🍩
